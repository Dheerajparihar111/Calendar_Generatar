# 📅 Calendar Generator 

A professional **Command Line Calendar Generator** built in **Java**, featuring a visually rich **boxed UI grid**, **multi-event management**, and **Indian holiday integration**.

This project simulates a real-world calendar system directly in the terminal, combining **clean design, structured logic, and user interaction**.

---

## 🚀 Features

### 📆 Calendar System

* Generate monthly calendar
* Correct day alignment using date algorithms
* Leap year handling

### 🎨 Visual Enhancements

* Colored CLI output
* 🔴 Sundays highlighted in red
* 🟢 Events highlighted in green
* 🔴 Holidays highlighted in red
* Clean grid-based calendar display

### 📝 Event Management

* Add multiple events per day
* Edit specific events
* Delete specific events
* Event marking (`*`) on calendar

### 🇮🇳 Indian Holidays

* Predefined national holidays
* Automatically marked on calendar
* Displayed separately for clarity

---

## 🛠️ Technologies Used

* ☕ Java (Core)
* 💻 CLI (Command Line Interface)
* 📚 ArrayList (dynamic event storage)
* 🗂 HashMap (holiday mapping)
* 🎨 ANSI Escape Codes (color output)
* 📐 Zeller’s Congruence (day calculation)

## 📌 How It Works

* Calendar is generated using mathematical date formulas
* Events are stored using `ArrayList[]` (multiple events per day)
* Holidays are stored using `HashMap`
* Grid UI is printed using formatted output
* Colors are applied using ANSI escape codes
